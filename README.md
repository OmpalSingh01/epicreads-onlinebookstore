📚 Epic Reads

Epic Reads is a full-stack online bookstore built with the MERN stack. It provides users with a seamless platform for browsing, purchasing, and managing books, with a responsive design optimized for all devices.

🚀 Live Demo: [Epic Reads on Vercel](https://epicread1.vercel.app/),
              (https://epicread1-ompalsinghs-projects.vercel.app/)


🛠️ Tech Stack

Frontend: React, Tailwind CSS, DaisyUI

Backend: Node.js, Express.js

Database: MongoDB

API Testing: Postman

Deployment: Vercel (Frontend), [Render(Backend)]

🎯 Project Objective

The goal of Epic Reads is to design and implement an online bookstore that:

Facilitates efficient browsing, purchasing, and management of books.

Provides a responsive and user-friendly interface with Tailwind CSS and DaisyUI.

Ensures cross-device compatibility for desktop, tablet, and mobile users.

✨ Features

📖 Book Catalog – Browse a wide selection of books by category.

🛒 Shopping Cart – Add, remove, and manage items before checkout.

🔍 Search & Filter – Quickly find books by title, author, or genre.

👤 User Authentication – Secure sign-up and login functionality.

💳 Purchase Management – Streamlined checkout process.

📱 Responsive UI – Works seamlessly across devices.

⚙️ Getting Started

Follow these steps to set up the project locally:

1️⃣ Clone the Repository
git clone https://github.com/OmpalSingh01/epicreads-onlinebookstore.git


2️⃣ Install Dependencies

For both client and server:

npm install

3️⃣ Setup Environment Variables

Create a .env (backend)  file in the server directory and add:

PORT=4001
MONGO_URI=your-mongodb-connection-string (Make it on MongoDb Atlas)

RZP_KEY=your-razorpay-key(Payment Gateway)


4️⃣ Run the Application

⚙️Start backend:

npm start


🌐Start frontend:

npm run dev
